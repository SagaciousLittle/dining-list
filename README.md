# 这是一个适用于公司点餐接龙的小程序，方便各位同学报名与统计~
## 服务端
[dining-list-server](https://github.com/SagaciousLittle/dining-list-server)
## 计划功能
- 报名
- 统计
- 排序
- PC端导出文档
- more...  
## 目录结构
- miniprogram
  - api 接口定义
  - pages 页面路径
  - utils 公共方法
## 注意事项
- 启动项目后需要配置：详情 - 启动自定义处理命令 - 编译前/预览前/上传前预处理 => ```npm run tsc```
